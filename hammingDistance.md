# 问题分析 #

    两个整数之间的汉明距离指的是这两个数字对应二进制位不同的位置的数目。

    给出两个整数 x 和 y，计算它们之间的汉明距离。

    注意：0 ≤ x, y < 231.


# 代码实现 #
```C
int hammingDistance(int x, int y) {
    int res=0,i;
    for(i=0;i<32;i++)
    {
        if((x&(1<<i)) ^ (y&(1<<i)))//按位分别取出两个数对应位上的数并异或
        {
            res++;
        }
    }
    return res;
}
```
# 总结体会 #
    汉明距离是指二进制数的不同位的个数。按位分别取出两个数对应位上的数并异或，异或是相同为

    0，不同为1，计算出1的个数，即汉明距离。
