# 问题分析 #

    给一个非负整数 num，反复添加所有的数字，直到结果只有一个数字。

    例如:

    设定 num = 38，过程就像： 3 + 8 = 11, 1 + 1 = 2。 由于 2 只有1个数字，所以返回它。 

# 代码实现 #
```C
int addDigits(int num) {
    int sum=0;
    int d;
    while(1)
    {
        while(num)
        {
            d=num%10;
            sum=sum+d;
            num=num/10;
        }
        if(sum<10)   
        {
            break;
        }
        num=sum;  //相加结果不是一个数字，继续重复相加操作
        sum=0;
    }
    return sum;
}
```
# 总结体会 #
     采用while循环，将给定数字的所有数字进行相加，若只有一个数字即小于10，则输出相加结果；

     若大于10，则继续重复相加操作，直到最后只有一个数字为止。
