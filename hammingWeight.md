# 问题分析 #

    编写一个函数，输入是一个无符号整数，返回的是它所有 位1 的个数（也被称为汉明重量）。

    例如，32位整数 '11' 的二进制表示为 00000000000000000000000000001011，所以函数返回3。


# 代码实现 #
```C
int hammingWeight(uint32_t n) {
    int i,res;
    for(i=0;i<32;i++)
    {
        if(n&(1<<i))
        {
            res++;
        }
    }
    return res;
}
```
# 总结体会 #
      按位取二进制数对应位上的数，如果为1，个数就加1，最后返回1的个数，即为汉明距离。
